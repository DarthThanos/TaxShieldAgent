# App Icon Required

Stripe requires a 256x256 PNG icon for the App Marketplace listing.

Requirements:
- 256x256 pixels
- PNG format
- No rounded corners (Stripe applies these automatically)
- Should be the TaxShieldAgent shield logo
- Brand colors: indigo (#6366f1) on white or dark background
